# React+AntdMobile

## 本项目使用Create React App进行初始化

## 如何使用？
### 从项目源码开始构建
#### 1.git clone https://github.com/hubtoks/DayBill.git   
#### 2.cd DayBill
#### 3.npm i 安装依赖，如有报错，请使用npm i --force进行安装
#### 4.npm run serve启动数据mock，模拟后端数据
#### 5.npm start启动项目，初次启动可能较慢，请耐心等待构建完成后，访问http://localhost:3000即可
